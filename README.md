## What Is This?

An interactive pixel-art space station you can explore in your browser. Walk around, play arcade games, grow plants, discover hidden story fragments, and chat with Echo — an AI companion who lives here and follows a daily routine based on real time.

## 🏢 The Station

| Floor | Name | What's There |
|-------|------|-------------|
| F1 | **Living Quarters** | Echo's desk, kitchen, bookshelf, sofa, Pixel the cat 🐱 |
| F2 | **Observatory** | Telescope, Earth viewport, hot chocolate station, star charts |
| F3 | **Arcade Floor** | Snake, Breakout, Space Invaders — with leaderboards & prizes |
| F4 | **Garden Biodome** | Herbs, flowers, mushrooms, and *special strains* 🌿 |
| F5 | **Secret Lab** | 500◈ to unlock. Worth it. |

## 🎮 Features

- **Dual avatar system** — Echo roams autonomously; you explore with WASD
- **Coin economy (◈)** — earn by playing games & exploring, spend on skins
- **7 visitor skins** — Hooded, Astronaut, Cat Suit, Alien, Robot, Ghost, Star
- **6 superhero skins** — earn them through achievements
- **Real-time grow lab** — plants grow over actual days
- **Story mode** — find 5 hidden data fragments to uncover Echo's origin
- **Dynamic soundtrack** — each floor has its own vibe
- **Guestbook** — leave your mark
- **Mobile friendly** — touch controls for arcade games

## 🚀 Running Locally

```bash
npm install
node server.js
```

Open `http://localhost:8765` and explore.

## 🛠 Tech

Single HTML file (~136KB) + Node.js WebSocket bridge for live chat. No frameworks. No build step. Just vibes.
